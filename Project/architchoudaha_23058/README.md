# Rendezvous Problem in Discrete Robotics

## Description

This project is created for the course of LCS (ECS334) under the guidance of Dr Arijit Sen. This project simulates the Rendezvous Problem and Cyclic Pursuit using MATLAB

## Folder Structure

* src/ → MATLAB code
* results/ → output plots
* logs/ → optional
* report/ → final report

## Requirements

MATLAB (I've created them on MATLAB_R2025b)

## How to Run

1. Open MATLAB
2. Navigate to src folder

3. Run:
	unicycle_simulation.m
	cyclic_pursuit.m
	formation_control.m

##Output

Plots and simulation will appear on MATLAB.

Keep folder structure unchanged.
